<h2>Como executar o projeto</h2>

```bash

# clonar repositório

git clone git@github.com:Paulo-cds/trequinhos-conexao-api.git

# entrar na pasta do projeto

cd projeto

# executar o projeto

yarn start

o projeto será executado no localhost:3000

```
